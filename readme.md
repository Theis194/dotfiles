# Dotfiles

## Setup

### 1. Clone

```bash
git clone --recurse-submodules https://github.com/yourusername/dotfiles.git ~/dotfiles
```

### 2. Change directory

```bash
cd ~/dotfiles
```

### 3. Bootstrap

```bash
bootstrap.sh
```
